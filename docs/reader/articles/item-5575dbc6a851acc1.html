越南企業在取締盜版後轉向開源軟體

越南首起軟體盜版刑事案件，正促使技術人員與企業放棄未授權的 Windows 與 Office 安裝；開源軟體也越來越成為首選的合法替代方案。 越南首起全國性、涉及未經授權使用受著作權保護電腦軟體的刑事案件，正在加速企業轉向開源軟體替代方案。技術人員與企業也開始遠離盜版 Windows 與 Microsoft Office 安裝。 6 月 11 日，富壽省公安廳調查警察機關依《刑法》第 225 條立案偵辦刑事案件，這也是越南首件與軟體著作權侵害相關的刑事起訴案件。 調查人員表示，嫌疑人提供預先安裝 Microsoft Windows 與 Microsoft Office 的電腦，而這些軟體顯示出透過破解程式、盜版產品金鑰，以及用來繞過著作權保護機制的啟用工具進行未授權啟用的跡象。 這波取締正在改變長久以來的軟體安裝慣例。過去承接 Windows 安裝的自由工作者與電腦維修店，越來越常拒絕涉及未授權軟體的合約，並轉而鼓勵客戶購買正版授權，或採用開源替代方案。 曾專門處理網咖系統與軟體部署的前電腦技術人員 N.A.D. 說：「對於預算有限、難以購買授權的客戶，特別是網咖，我通常會建議他們改用開源軟體，因為現在已經有很多很好的替代方案。」 胡志明市一家電腦維修店老闆 N.T.A. 表示，他的團隊現在會建議客戶使用正版授權軟體，也會拒絕替不願遵循規範的組織提供維護服務。 當局表示，正版 Windows 與 Office 授權每台設備價值約 400 萬至 900 萬越南盾。由於查獲數百台侵權電腦，對智慧財產權權利人的估計損失達數百億越南盾。官員並補充，軟體盜版會削弱研發、侵蝕智慧財產權保護，並扭曲公平競爭。
