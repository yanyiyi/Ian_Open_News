Agentic AI Foundation（AAIF）

  • goose 現已內建由 llama.cpp 驅動的本機推論功能｜程式碼完全不會離開裝置。無須 Docker、Ollama 或 API 金鑰，模型可完全離線執行。對於在敏感環境或實體隔離系統中工作的團隊而言，這是意義重大的一步。
  • goose v1.26 推出 Computer Controller 擴充功能 | 以 Peekaboo 為核心重新打造。Peekaboo 是一款 macOS 命令列工具，讓 goose 能以「先看、再點擊、接著輸入」的循環執行桌面自動化。它可處理含標註的螢幕截圖、辨識 UI 元素、輸入文字、捲動、拖曳、選單與對話框。現在，goose 已能在沒有完善 API 的桌面與網頁應用程式之間自動執行工作流程。前往 goose 部落格閱讀更多 >>以及在 GitHub 查看 Peekaboo >>
  • 全新社群工具著手解決 MCP 在真實環境中最棘手的問題之一 | MCP Debugger 的開發者花了數個月處理正式環境中的授權流程、安全性邊界案例與用戶端行為。區域測試通過了，冒煙測試也通過了，但真正的代理程式透過 HTTP 連線時，卻暴露出難以發現、更難修復的問題。MCP Debugger 正是為了彌補這道落差而打造。閱讀完整部落格文章 >>
  • Solo.io 將用於正式環境代理可靠性的 AgentEvals 開源 | 此專案會根據 OpenTelemetry 追蹤資料評估代理程式的行為，讓團隊無須重新執行成本高昂的大型語言模型（LLM）工作階段，就能為工具呼叫、執行路徑、回應品質與回歸問題建立基準。它支援離線與即時評估、黃金評估集、內建評估器、命令列與網頁工作流程、CI/CD 應用，以及可透過對話檢視代理程式工作階段的 MCP 伺服器。AgentEvals、Solo.io、GitHub >>
  • MCP Apps 登上行動版 Claude | Anthropic 已在 iOS 與 Android 版 Claude 推出 MCP Apps 支援，將桌面版與網頁版的互動式 iFrame 元件模式延伸至手機，讓第三方工具能在行動裝置的代理程式對話中提供帶有自身品牌、可供點選操作的使用者介面。
  • Cloudflare 推出網站 Agent Readiness Score（代理程式就緒度評分）｜全新的 isitagentready.com 掃描工具會檢查網站是否提供代理程式所需的標準支援，包括探索內容、遵循機器人存取權限、讀取 Markdown、呼叫 API、尋找 MCP 伺服器卡片，以及支援新興商務流程。Cloudflare 也將這些檢查功能加入 URL Scanner 與 Radar，並讓掃描工具本身可作為無狀態 MCP 伺服器呼叫，為團隊提供一條具體的正式環境導入途徑，使文件與服務能供代理程式使用。Cloudflare >>
  • 三場指標性的 AGNTCon + MCPCon 活動將於今年秋季登場｜東京（9 月 10 至 11 日）、阿姆斯特丹（9 月 17 至 18 日）及聖荷西（10 月 22 至 23 日）。這些活動將匯聚正在建構正式環境代理式系統的組織，共同推動技術跨越概念驗證階段。查看完整的 2026 年活動一覽 >>
