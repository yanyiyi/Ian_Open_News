有人偷用你家 Wi-Fi 的 3 個明顯跡象，以及阻止方法

監控正持續滲入我們的日常生活。街上到處有臉部辨識攝影機；我們在線上也不斷被追蹤；參加社交活動時，也很少不會有人拿著相機對著我們拍。然而，如果這種監控進到家裡，可能更令人不安。 你是否曾注意到有陌生裝置連上家中網路？或是頻寬突然快速下降？如果有人潛伏在你的 Wi-Fi 上，那就是嚴重侵犯隱私；更不用說你正在為這項服務付費，對方卻在占用你的資源。 如果有人取得你的 Wi-Fi 存取權，他們能做的不只是偷用你付費購買的頻寬；他們也可能危害你的隱私與安全、竊取你的資料，甚至讓你的網路活動和他人的非法線上行為被牽連在一起。 幸好，要辨識網路上的陌生人並把他們踢出去，通常並不困難。以下說明如何偵測他們、請他們離開，並避免他們再進來。

留意這三個警訊

1. 你的網路變慢

網路連線突然從快速變得遲緩，或出現意外斷線與延遲，原因有很多。合理的中斷原因包括路由器硬體故障、外部線路受損、訊號強度不足，或網路服務供應商（ISP）端的問題，例如區域性斷線。軟體方面，如果使用錯誤的 VPN 或 VPN 伺服器來保護連線，也可能導致嚴重降速。 不過，網路變慢也可能是因為你的網路上有不速之客正在偷用頻寬。連上同一個 Wi-Fi 存取點的裝置越多，流量需求就越高。若有人正在高畫質串流、線上遊戲，或透過 torrent 下載大型檔案，情況尤其明顯。 要分辨網路變慢是合理原因造成，還是被搭便車的人導致延遲，可能並不容易。你可以先查看 ISP 的服務狀態是否異常、到 downforeveryoneorjustme.com 檢查線上服務是否中斷，並檢查硬體，包括線材與接頭；因為線路鬆脫也可能造成寬頻問題。

2. 網路上出現未知或不認得的裝置

如果有人未經允許使用你的 Wi-Fi，他們必須連上一台裝置，例如智慧型手機、PC、喇叭或智慧家庭設備。 你也可能會遇到網速低、流量負載高、影片緩衝，甚至自己的裝置毫無預警斷線。當太多裝置，不論是否獲得授權，都在爭奪同一份頻寬時，網路就可能不堪負荷。 另一個跡象是路由器顯示的即時流量與連線狀態。例如，硬體上的燈號閃爍可能代表網路活動，即使你並沒有預期會有流量。不過，在認定是未知裝置造成前，還是應先查閱路由器手冊，確認是否有其他原因。

3. 路由器停止運作，或你失去 Wi-Fi 存取權

如果 Wi-Fi 突然停止運作，可能代表有人未經允許動了你的設定。若你的網路會在特定時間開關，例如你正在播放音樂時，更值得懷疑。 要維持對你 Wi-Fi 熱點的存取，對方必須離你很近。如果你和鄰居或家中某人關係不太和睦，就始終存在有人故意干擾你 Wi-Fi 的風險。例如，若有人已經取得你的 Wi-Fi 與閘道器存取權，就可以隨時中斷你的連線。 此外，如果 Wi-Fi 密碼被更改，而你被踢出網路，很可能有人已經取得控制權，並在未經允許的情況下改掉密碼。 不過，你也應該記得，問題可能來自硬體故障，而不是惡意人士。路由器會隨時間故障、ISP 服務可能不穩，甚至天氣狀況也可能影響連線。但如果服務中斷有固定模式，例如每天某個特定時間發生，就可能是刻意造成的。

別人怎麼會未經允許使用我的 Wi-Fi？

最簡單的答案通常就是正確答案；在 Wi-Fi 這件事上，通常問題出在密碼。 可能是你的密碼太弱、太容易猜，或是在未經你允許的情況下被分享出去。也可能有人對某些 Wi-Fi 網路進行暴力破解或字典攻擊，執行程式測試字母、數字、片語和特殊字元的組合，藉此猜出密碼並闖入。 Wi-Fi 安全性本身也是問題。Wi-Fi 標準包括 WPA、WPA2 與 WPA3。你可以到路由器的無線設定中查看 Wi-Fi 使用的通訊協定：在 macOS 上，按住 option 鍵並點擊 Wi-Fi 圖示；在 Windows 上，前往「設定」、Wi-Fi，接著查看安全性分頁。 標準越新，攻擊者越難入侵。但如果密碼很差或網路設定錯誤，無論使用哪一種協定，都不是完全不可能被突破。過時的路由器韌體也可能讓你的 Wi-Fi 面臨風險，因為其中可能含有可被利用的漏洞。你可以在路由器閘道器中查看硬體的韌體版本，確認它是否會自動更新，或是否需要透過網頁手動更新。

如何找出網路上的可疑裝置？

要確認是否有人未經允許連上你的 Wi-Fi 網路，最簡單的方法是進行網路掃描。 第一步是透過瀏覽器進入路由器的閘道器。你的路由器製造商或 ISP 應該會提供需要造訪的位址；如果不確定，可以試試 192.168.0.1 或 192.168.1.1。另一個方法是打開智慧型手機的 Wi-Fi 設定，點選家中的 Wi-Fi 熱點，進入設定，並在可用時選擇「管理路由器」；這應該會直接帶你到閘道器。 透過瀏覽器進入閘道器時，你可能會看到連線不是私密連線的警告，接著需要用集線器或路由器密碼登入。登入後，依 ISP 而定，你可能可以看到網路上列出的所有裝置。 或者，你也可以使用專門的網路掃描工具，找出並顯示所有裝置與 IP 位址，再檢查是否有陌生項目。

還能怎麼確認是否有人闖入 Wi-Fi？

執行測速： 如果可以，在一天中的不同時間測速，看看是否能找出模式。例如，如果你的起始速度本來就偏低，而有人下班回家後開始用 Netflix 看 4K 串流，你可能會立刻注意到服務品質下降，甚至完全無法載入內容。 檢查連線紀錄： 熟悉技術的人可以使用 Nmap 與 Fing 等工具產生紀錄，這些是專門用於網路與 Wi-Fi 對應的工具。它們可能讓你了解有哪些裝置正在連線、連了多久，以及何時連線。 觀察你的周遭環境： 例如，如果鄰居正在大聲播放 Spotify 播放清單，而你拔掉路由器後音樂突然停止，你可能就找到了 Wi-Fi 服務斷斷續續的原因。

如何阻止網路入侵者？

更改密碼： 如果有人透過猜出密碼悄悄使用你的 Wi-Fi，改密碼就是最快也最簡單的踢人方式。你可以依前述方式進入路由器閘道器來更改密碼。 選擇更好的密碼： 不要依賴預設密碼來保護 Wi-Fi，也不要使用容易記住或方便分享給訪客的密碼。建立一組複雜密碼。雖然很麻煩，但請花時間把所有裝置重新連上 Wi-Fi；因為改密碼代表使用舊憑證登入的裝置會被移出網路。雖然我們不建議把重要線上帳號的密碼寫下來，但家用 Wi-Fi 的密碼寫下來並放在家中安全處，風險相對較低。 檢查管理員密碼： 你也應該檢查路由器或 ISP 閘道器的管理員密碼。如果仍是預設組合，例如 admin/admin，請改成更強的密碼，並妥善保存在安全地方。 檢查 Wi-Fi 安全性，必要時升級： 如果你使用的是過時或舊式通訊協定，例如 WPA，它本身就不夠安全，無論你做什麼都可能容易被破解。這種情況下，你需要升級通訊協定。通常你可以自行進入路由器閘道器，在安全性或無線設定中變更加密標準。之後你可能需要設定新密碼，並重新連接各項裝置。如果可以，WPA3 是最佳選擇。 考慮設定訪客網路： 如果你偶爾需要分享 Wi-Fi，可以設定訪客網路。這會建立一個與主要網路和裝置分開的存取點，讓你對誰能連線、可以連多久，以及能使用多少頻寬，有更多控制權。
