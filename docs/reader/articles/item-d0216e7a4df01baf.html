授權與合規實驗室：傳遞知識、捍衛自由

大家好，感謝你投入自由軟體。現在，我們比以往任何時候都更需要理解自由軟體理念的人。

我是 Krzysztof Siewicz（Kris），擔任自由軟體基金會（Free Software Foundation，FSF）的授權與合規經理。我與 FSF 著作權及授權專員 Craig Topham，以及一群投入的志工，共同營運 FSF 的授權與合規實驗室。我想談談我們近期取得的成果與未來規畫；要實現這些計畫，我們仰賴像你一樣支持自由軟體的人協助。

除了持續捍衛 GNU 通用公眾授權條款（GPL）與電腦使用者自由等日常工作外，我們近期也將許多心力投入機器學習。我們密切關注人們如何尋求以保障自由的方式使用機器學習模型，包括自行起草授權條款，以及如何將機器學習整合至自由軟體開發。我們收到許多關於軟體自由未來的問題。在機器學習領域，我們參與了 FSF 的「自由機器學習組合包定義」（Free/Libre Machine Learning Bundle Definition）工作小組。該小組正在敲定正式發布的細節，並為 GNU 專案起草接受大型語言模型（LLM）生成程式碼的政策。

以下是實驗室過去幾個月為軟體自由所做的部分重點工作：

  • GPL 合規：我們處理 GPL 違規者時所遵循的原則之一是保密，因此我只能透露，我們正與數家散布者聯繫，而持續交涉已促使對方公布一些可能符合要求的原始碼。不過，我們仍須確認這些原始碼是否完整，並附有完整的安裝資訊。除了處理侵犯 FSF 著作權軟體的已知違規者，我們也持續審查每天收到的 GPL 違規通報，教導通報者如何蒐集證據、聯絡著作權人，並請他們與律師合作取得原始碼——這是使用者應有的權利。過去六個月，我們平均每月處理 13 件授權違規通報。
  • GPL 管理與維護：我們持續蒐集容易造成混淆的 GPL 使用案例，並與那些在 GPL 中加入易混淆附加條款的人合作，確保使用者清楚知道自己可以仰賴授權條款所保障的自由。這類容易造成混淆的修改中，有一大類是要求標示作者姓名的條款。作者署名是自由軟體開發的支柱之一，但署名要求不應被用來限制使用者自由。因此，我們整理了現有的授權要求與立法意旨資訊，發布關於 GPL 相容之合理法律聲明與作者署名的指引。我們也持續直接協助希望為 GPL 加入附加條款的專案，確保其修改方式符合 GPL。
  • 協助 GNU 專案：FSF 的宗旨是服務整個自由軟體社群及所有軟體使用者。我們尤其高興能為 GNU 軟體套件的開發者與維護者提供實際協助。我們已回覆多項與不同 GNU 套件有關的問題，包括協助一個新套件獲 GNU 接納。我的同事 Craig Topham 一如往常忙於處理著作權讓渡事務。我們也很自豪地宣布，近期已更新著作權讓渡範本，明確確保其在德國及法律制度相近的司法管轄區也能正常適用。感謝 Till Jaeger 無償提供協助！
  • 邀請志工協助審查授權條款：為了讓志工更容易協助處理積壓的授權條款審查工作，同時讓這項流程成為社群的學習機會，我們已在 LibrePlanet 授權志工入口網站建立 Wiki，供大家進行初步的授權條款研究。
  • 回答授權問題：在傑出授權志工的協助下，我們持續解釋 GNU 授權條款預期應如何適用，平均每月回答約 22 個問題，其中許多相當複雜。這項工作高度仰賴外界支持，因為我們免費向所有自由軟體開發者提供這項服務。
  • 重新授權與相容性：例如，licensing@fsf.org 經常收到的一類問題，是如何組合採用不同授權條款的程式。我們很高興能與長期志工、傑出的自由軟體授權專家 Yoni Rabkin 合作，發布其文章〈重新授權與相容性的差異〉的修訂更新版。
  • 審查授權條款：許多人在想使用的程式中看到某項授權條款，卻無法判斷它是否屬於自由軟體授權，或能否與其他預計組合使用之程式的授權條款相容，因此向我們提出許多問題。我們的資源有限，審查授權條款時經常捉襟見肘；但近期仍檢視了一些我們認為格外重要，或容易令自由軟體社群混淆的授權條款，例如 Source First License v1.1 與 RAIL 授權條款。
  • 防止授權條款氾濫：我們也會收到自行起草新授權條款者的問題。由於授權條款氾濫會造成問題，我們強烈建議他們重新考慮。我們也向他們說明，起草授權條款是一項困難的工作，必須有律師參與，並充分理解該授權條款預計處理的問題。我們會說明如何運用現有的自由軟體授權條款處理相關問題。若有人仍堅持以某一版本的 GPL 為基礎制定自己的授權條款，我們會確保他們遵守我們的規定。
  • 培訓新授權志工：我們隨時歡迎大家投入協助，也努力讓授權志工服務成為學習機會。成為授權志工須通過 GPL 測驗、接受所用工具的操作訓練、熟悉 GPL 常見問答及其理由，並簽署保密協議。我們隨時歡迎新成員加入；FSF 員工及資深志工都會提供指導，因此參與者也能有所收穫。不過，能及時回答更多問題，當然才是整個自由軟體社群獲得的最大利益。來自奧地利、經驗豐富的法律顧問與調解人 Melanie Carmen Punz 最近加入了團隊。她的協助已經發揮關鍵作用，不只幫忙回答授權問題，也投入 GPL 的管理與合規工作。感謝 Melanie，也感謝你提供支持，讓我們能與更多志工合作。

我們知道，不是每個人都有能力捐助；但如果情況允許，請加入 FSF 成為贊助會員，支持我們的工作。實驗室能夠持續運作，很大程度仰賴像你一樣的支持者捐款。贊助會員資格能展現我們可以倚靠的堅定支持。捍衛你的自由時，FSF 絕不妥協。若沒有大家持續投入，我們已取得的進展將面臨危機，軟體自由也可能從今日的現實退化為遙不可及的願望。請協助我們繼續保護軟體自由，並在全球爭取更多支持。今年夏天加入的每一位會員，都能讓我們朝新增 175 名會員的目標更進一步。贊助會員也能享有所有會員福利，包括商品折扣、16GB 可開機會員卡，以及使用贊助會員視訊會議伺服器。

當你承諾維護使用者自由時，所承諾的不只是自由軟體，也是為所有人打造一個自由的社會。只要攜手合作，我們就有力量保護軟體自由。
