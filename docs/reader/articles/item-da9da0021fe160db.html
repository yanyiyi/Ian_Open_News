用 LoRa Mesh 找物的 SenseCAP T1000-E 追蹤卡實測

這篇是 SenseCAP T1000-E 追蹤卡的實測心得。作者把它放在 AirTag 與其他藍牙追蹤器的脈絡下比較，重點在於 T1000-E 不只靠手機藍牙，也能連到 LoRaWAN、Meshtastic、Amazon Sidewalk、Helium 等網路；其中 Meshtastic/LoRa mesh 的賣點是低功耗、長距離、可離線、點對點，不一定仰賴電信或大型平台公司的基礎設施。作者肯定它的定位能力、戶外耐用性與 GPS 表現，也提醒實務缺點：電池續航只有幾天、專用磁吸充電線容易脫落、App 說明與介面體驗不夠好。整體而言，它可作為開源 mesh 技術進入消費硬體的觀察案例，但文章本身偏產品評測，不是深入的開放技術分析。

這款藍牙追蹤器用 LoRa Mesh 找東西，精準度讓我難以置信

我以前說過，現在還是要再說一次：這類追蹤標籤對我來說真的徹底改變了使用體驗。能夠在世界各地精準定位鑰匙、錢包這類物品，替我省下了無數心痛、頭痛和浪費掉的時間。第三方標籤越來越多，也代表現在已經有適合各種用途的追蹤標籤。 但這些標籤有一個缺點：它們都仰賴行動網路和 Wi-Fi 網路。如果有一種追蹤標籤，可以在自己的網路上運作，不必依靠那些由市值數十億美元企業建起來的網路呢？這正是這款產品提供的能力。 好，那它到底怎麼運作？和多數產品一樣，T1000-E 有很多我喜歡的地方，也有幾點我不喜歡。不過在繼續往下講之前，我得先提醒你，這張卡很可能會成為讓你跌進 LoRa Mesh 廣大世界的入門毒物。LoRa 是 Long Range Radio 的縮寫，是一種長距離無線電協定，可以用來建立 Mesh 網狀網路。 什麼是 Mesh？它是一種開源、離網、去中心化的網狀網路，設計用來在小型、低功耗裝置上運作。它不需要基地台，也不需要網際網路。這是一套完全獨立、點對點的無線電系統。如果你有點像我，喜歡鑽研奇奇怪怪的東西，很快就會深陷各種 Mesh 收發器，開始架設自己的節點。為什麼？因為你可以。 注意：請確認你買的是 T1000-E，而不是 A 或 B 版本。只有 E 版是設計來搭配點對點 Mesh 使用；A 版和 B 版則會使用、也需要各自的網路閘道器。 回到 T1000-E。這是一張卡片大小的追蹤器，和其他同類產品差不多。它的大小像信用卡，厚度則約等於一疊卡片。裡面有一顆 700 mAh 電池，足以讓這張卡運作幾天；另外也塞進了所有用於通訊與定位的技術元件。 它有一個音量很大的蜂鳴器、一盞 LED 狀態燈，以及一顆按鈕，可用來開關裝置，並控制藍牙等功能。

為末日情境打造

外殼具備 IP65 防塵防水等級，所以帶到戶外使用沒有問題。我很喜歡這點，但為了達到這樣的防塵防水能力，製造商選擇在背面採用磁吸充電板，並透過 pogo pin 彈簧針接點充電。 這不只代表每次充電都需要專用線材，也代表如果你想在移動中替這張卡充電，例如用某種行動電源之類的裝置，它很可能會鬆脫，然後停止充電。我自己想過用橡皮筋或熱熔膠來解決，但我還是希望有更好的方式，能讓接頭穩穩固定住。

它是很棒的追蹤器

預設情況下，你需要下載 SenseCraft app 來設定與控制 T1000-E。這個 app 還可以，但稱不上出色。說明有時候有點模糊；偶爾你會看到中文段落，而且它還會試圖向你推銷更多東西。對一個我已經買下來的產品來說，我很討厭 app 裡還出現這種推銷。 我的建議是，如果你卡住了，可以去看內容完整的支援文件 wiki，裡面也有影片；或者到 Reddit 上很棒的 r/meshtastic 社群看看。 如果你只是想把它當作追蹤器使用，那你基本上已經完成所有需要做的事了。app 裡有地圖，可以顯示這張卡的位置；你也可以在外出時透過分享位置，把自己的所在位置廣播給其他人。 這張卡會透過藍牙連接智慧型手機上的 app，但它也能連上 LoRaWAN、Meshtastic、Amazon Sidewalk 和 Helium 網路。 如果你住的地方有人在使用 Mesh，那會是一個很好、而且能直接運作的選項。你也可以選擇在住家或工作場所周圍架設節點，讓自己使用，同時也為所有人擴展 Mesh 覆蓋範圍。 我發現 Amazon Sidewalk 也是一個非常好用、可行的選項，尤其是在建成區。畢竟外面有很多相關裝置，所以你可以利用的 Mesh 規模其實相當大。選項很多。 沒錯，這也可以在那些末日情境中派上用場：停電、行動網路沉寂，殭屍開始在外面晃來晃去的時候。我把它放在口袋、包包裡，也曾綁在背包外側；即使在條件不太理想的情況下，GPS 接收器的表現仍然很出色。卡片上也有吊繩孔，方便把它固定在物品上。

但還有更多……多很多！

T1000-E 的能耐還不只這樣。它遠遠不只是一張追蹤卡。它是一個功能完整的 Mesh 發射器，可以在其他裝置之間收發訊息。如果你想深入玩這件事，就會需要另一台 Mesh 裝置，例如另一個追蹤器，或其他相容裝置。也就是從這裡開始，你會一路滑進對 Mesh 著迷的坑裡。 你也可以更深入地研究這張卡。例如，你可以把 Seeed Studio 載入 T1000-E 的韌體清掉，並使用線上燒錄工具把原版 Meshtastic 韌體安裝進去。這樣一來，你就可以用 Meshtastic app 控制這張卡；我發現這樣好用得多，尤其是當你想更深入探索這張卡的能力時。 這已經超出本文能詳細說明的範圍，但如果你想找個起點，原文提到的那支影片大概會讓你正式入坑。

ZDNET 的購買建議

這款產品大約要價 50 美元。你有時候會看到它賣得貴很多，但請避開那些價格。通常只要等個一天左右，價格就會降下來。 如果只把它當成純粹的追蹤卡，這個價格相當不便宜；但當你意識到它也能作為訊息用戶端，並且可以免費收發訊息時，這個價格就開始顯得合理一些。 Mesh 很適合離網通訊，也很適合郵輪這類地方，因為船公司通常會把 Wi-Fi 收費訂得非常昂貴。
