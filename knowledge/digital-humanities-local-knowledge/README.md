# 數位人文與在地知識建構

這條主線收錄文化記憶、地方知識、數位典藏、博物館、檔案、社群共筆、在地媒體與公共史學相關資料。

## 工作區

- `research/`：來源蒐集、背景資料、舊資料對照。
- `briefs/`：可 review 的知識 brief。
- `published/`：已定稿或已發布的整理。

## 收錄判斷

一筆資料值得進一步處理，通常至少符合其中一項：

- 反映地方知識如何被蒐集、詮釋、保存或再利用。
- 涉及文化記憶、數位典藏、地方資料庫、博物館或檔案。
- 呈現社群、居民、研究者、公共部門之間的知識協作。
- 能連到開放文化、公共資料、數位近用或知識治理議題。

## 審查底線

- 摘要必須忠於原文，不把「我們的解讀」寫成「來源的主張」。
- 保留來源、作者、組織、地點、時間與 URL。
- 涉及族群、地方衝突、個人資料或敏感記憶時，必須在 `review.notes` 標風險。
- 不把地方或社群只寫成可被抽取的素材。
