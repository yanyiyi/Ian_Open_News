# License Pattern References

Open only the pattern that matches the current source.

- `google-docs-footer.md`: Google Docs or exported document footer wording.
- `ocf-explicit-attribution.md`: OCF-style explicit license and attribution blocks.
- `openmuseum-license-link.md`: museum collection pages where the license is behind a linked rights statement.
- `tcmb-split-license.md`: pages where text, images, metadata, and logo have split rights.

Default to `未明確標示` when none of the patterns fit or when the page does not name a specific license/version.
