# Google Docs Footer Pattern

Google Docs, slides, and exported PDFs may put rights information in a footer or final page.

Check:

- Footer text on every page.
- The final page or acknowledgements.
- Links embedded in phrases like `Creative Commons`, `CC BY`, or `授權`.
- Whether the footer applies to the whole document or only copied material.

If the document only says it uses Creative Commons without a named license/version, mark `未明確標示`.
