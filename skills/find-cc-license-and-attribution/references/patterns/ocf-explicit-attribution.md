# OCF Explicit Attribution Pattern

OCF pages and decks often state a license near the footer, README, or release notes.

Prefer explicit statements such as:

- `本作品採 CC BY 4.0 授權`
- `This work is licensed under CC BY 4.0`
- A direct Creative Commons license URL

Record OCF or the named project/person as the attribution target only when the page says so. If the page includes third-party images, split those rows instead of applying the page license to every asset.
