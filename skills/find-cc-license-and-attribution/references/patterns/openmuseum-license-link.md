# Open Museum License Link Pattern

Museum and collection sites often show a rights label beside a linked license or rights statement.

Check:

- Object metadata fields: rights, license, usage, credit line, provider.
- Linked license URLs.
- Whether the license applies to the image, metadata, object description, or whole page.

If a page links to a rights statement but the displayed object has no explicit reusable license, mark `未明確標示`.
